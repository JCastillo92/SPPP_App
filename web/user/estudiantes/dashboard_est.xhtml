Estado del Proceso

  • 1.- Ingreso Datos Basicos

    Realizar el ingreso de datos del estudiante como de la empresa en el Wizard

  • 2.- Descargar/Subir Oficio

    La carrera generará un oficio para la empresa el cual debe ser descargado por el estudiante y entregado en la empresa

    Como respuesta a este oficio la empresa debe emitir una carta de aceptación la misma que debe ser escaneada y subida al sistema

  • 3.- Ingreso Datos Carta Compromiso

    Llenar los datos solicitados en la carta compromiso acerca del estudiante, empresa y el proceso

  • 4.- Descargar/Subir Carta Compromiso

    Descargar la carta compromiso para proceder a que las firmen las autoridades necesarias

    La carta compromiso una vez firmada debera ser subida al sistema por el estudiante

  • 5.- Solicitar Inicio

    Descargar el documento generado automaticamente para solicitar el inicio formal del proceso

  • 6.- Descargar/Subir Solicitud

    Descargar la solicitud de inicio para proceder a que la firmen

    La solicitud inicio una vez firmada debera ser subida al sistema por el estudiante

  • 7.- Resolucion de Proceso

    Esperar por la resolucion que se otorgue al proceso

  • 8.- Visita completada

    Esperar por confirmación del tutor
